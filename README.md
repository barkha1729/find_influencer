# find_influencer
This repository contains implementation of PageRank algorithm or HITS algorithm which can be used to find the most influential node in the graph. Common usecases are in finding the social media influencers, highly talked TV shows, trending topics etc.

The code is compiled in terminal with command: python pg.py

For any particular dataset the NODES and EDGES has to be updates accordingly.
The file should be in the format (from node, to node)
The nodes are edges are generated a per the input file format given above.
